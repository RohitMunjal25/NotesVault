📝 NotesVault - Your Secure Digital Brain
NotesVault is a full-stack web application designed to help users organize their thoughts and ideas securely. The project focuses on a Clean UI, High Performance, and Robust Security.

🚀 Key Features
Secure Authentication: JWT-based login and signup system with 6-digit OTP verification via email.

Password Recovery: Integrated "Forgot Password" flow allowing users to reset passwords using a secure OTP.

Smart Dashboard: A modern masonry grid layout that automatically organizes Pinned and Unpinned notes.

Pin/Unpin Logic: Prioritize important notes by pinning them to the top of the feed.

Real-time Search: Instant filtering of notes by title or content using an optimized search bar.

Full CRUD Operations: Complete control to create, read, update, and permanently delete notes.

Premium Dark UI: A sleek, responsive interface designed for focus and productivity.

🛠️ Tech Stack
Frontend
HTML5 & CSS3: Custom layouts with Glassmorphism and CSS variables.

Vanilla JavaScript: Direct DOM manipulation and Fetch API for seamless backend communication.

Backend
Node.js & Express.js: RESTful API architecture.

MongoDB & Mongoose: Document-oriented database for flexible data storage.

JWT (JSON Web Tokens): Secure stateless authentication.

Nodemailer: Automated email service for OTP delivery.
